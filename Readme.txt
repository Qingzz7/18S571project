To Run Models:
1. Open R Studio
2. Open Models.R in Yuqing
3. Ensure source folder is completedata 
4.Random Forest and Xgboost are loaded from complete data. To run takes a while with randome forest

To Run data preparation script
1. Open R Studio
2. Open dataprep.R in Travis
3. Ensure source folder is raw data

To Run shiny
1. Open R Studio
2. Open shinyPlots.R
3. Ensure you have run the Models.R first
4. Run app
